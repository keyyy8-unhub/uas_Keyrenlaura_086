Nama Platform: MYKOS

🎯 Problem yang diselesaikan:

Membantu mahasiswa perantauan mengelola biaya operasional kos, memantau pengeluaran bulanan, mencari makan hemat di sekitar lokasi, dan memastikan tagihan penting tidak terlewat.

🌿 User Flow Ringkas:

  1. Landing Page → Login/Registrasi
  2. Dashboard Utama (Pilih Layanan: Atur Kos, Cek Makan, Laporan, Pengingat)
  3. Masuk ke Fitur Detail (Lihat info kos, peta kuliner, grafik pengeluaran, atau list tugas)
